Utiliser uniquement du HTML et du CSS pour créer un site internet permet de rester sur des choses très simples. Cependant, le HTML est un langage de balisage statique. Tout ce qui doit être affiché doit être écrit. S'il y a 138 éléments à afficher, il faut définir 138 éléments dans le fichier HTML, même s'ils ont tous un aspect identique. C'est assez frustrant en tant que développeur car on a l'impression de faire toujours la même chose et de répéter les actions inutilement. Il doit y avoir un moyen de réaliser un site sans dupliquer le code mais pour cela nous allons avoir besoin d'un langage de programmation complet : Le javascript.
